INSTALLATION
============

Unzip the contents to any directory, from now on refered to as "installDir".
Add "installDir/bin" to your PATH environmental variable.

Verify the installation. From the command prompt, enter:
$ quokka -version

Ensure that output matches the version you installed.

NOTE: When you run quokka for the first time, it will create a .quokka directory in your
$HOME directory (or ~/Library/Preferences/Quokka on OS X) and seed it with some default settings.

See the bundled manual (quokka-manual.pdf) or http://quokka.ws for more information.
