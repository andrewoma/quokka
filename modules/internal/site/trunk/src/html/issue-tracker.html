Issue Tracker

Quokka's issue tracker is split into two sub projects:

  • Core: Issues relating to the quokka core belong here, including bootstrapping, dependency handling, project and repository processing.
  • Contrib: Issues relating to all extensions belong here, including plugins, dependency sets, xml catalogues and archetypes.

Please search the existing issues before creating a new one. Also, if just have a question, please use the mailing list instead.
