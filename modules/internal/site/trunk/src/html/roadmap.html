Roadmap

The Quokka core would seem to be pretty much complete as Quokka now covers almost all of the use cases it set out to achieve. Community feedback is now required to validate this is true for other users, with a particular focus on making any changes to public APIs for other developers ASAP. This includes the plugin and repository APIs as well as the bootstrapping module.

Bugfixes will the highest priority, but after that next on the list include:

  • Adding support for the Scala Programming language
  • Adding support for the TestNG testing framework
  • Expanding the JEE plugin to support EJBs and EARs
  • Investigate IDE integration with Netbeans and Eclipse
  • Expand the developer documentation

Check out the Issue Tracker as well for information on outstanding tasks and priorities
